Почему можно доверять

Точный дедуп

Дубликаты — по SHA-256, похожие кадры — по перцептивному хешу. Ничего не теряется и не копируется дважды.

Атомарная запись

Восстановление после сбоя — архив никогда не окажется в наполовину записанном состоянии.

Оригиналы неприкосновенны

Программа не изменяет, не удаляет, не переносит и не переименовывает исходные файлы — только копирует.

Полностью офлайн

Ни один файл, ни один байт не покидает ваш компьютер — интернет для работы не нужен.

Наглядный отчёт после каждого прогона

Сколько фото, какой охват по годам, что нашлось повторного — одной страницей.

Пример на синтетических данных — реальный отчёт строится по вашему архиву.

Поддержать разработку — скоро

Способ поддержки пока настраивается. Программа бесплатна в любом случае.
